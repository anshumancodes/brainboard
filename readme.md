## brainboard
brainboard is a realtime collaborative whiteboard tool that lets you easily sketch diagrams that have a hand-drawn feel to them.

brainboard will be available as a standalone application at [brainboard.anshumancdx.xyz](brainboard.anshumancdx.xyz) and will be integrated inside mindvault app as a internal feature at [https://mindvault.anshumancdx.xyz/brainboard](https://mindvault.anshumancdx.xyz/brainboard)

### techstack
- nextjs 
- expressjs
- supabase (for auth and postgressql db) 
- websocket
